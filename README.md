# Vadim Filimonov's dotfiles

This is a repo for my dotfiles.

## Ubuntu

### Pre-installation

```sh
sudo apt install git ansible make
```

### Installation

```sh
git clone https://github.com/vadimfilimonov/dotfiles .dotfiles
cd .dotfiles
make install-ubuntu
```

## MacOS

### Pre-installation

```sh
brew install ansible
```

### Installation

```sh
git clone https://github.com/vadimfilimonov/dotfiles .dotfiles
cd .dotfiles
make install-macos
```
